About This Project

Learn about the methodology, dataset, and technology behind this heart disease analytics platform.

Project Overview

Heart disease remains one of the leading causes of mortality worldwide. Lifestyle changes, poor dietary habits, and physical inactivity continue to increase cardiovascular risk.

This project uses Tableau dashboards to transform raw healthcare data into meaningful insights. The goal is to identify patterns, highlight key risk factors, and support evidence-based decision-making for healthcare professionals and researchers.

Note: This platform is built for educational and analytical purposes, not for direct clinical diagnosis.

How to Use This Platform

  • Select any dashboard from the homepage gallery.
  • Use Tableau filters to explore specific demographics (age, gender, race).
  • Hover over charts for detailed values and trends.
  • Compare lifestyle and health indicators across patient groups.
